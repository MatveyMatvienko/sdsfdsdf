Connect to your server

To connect to your server, copy the server address and enter it in your Minecraft client, as a new server or with "Direct Connect". You can find the server address on the server page.

This address includes your server port if "Cracked" is enabled in the options. Read more details about cracked launchers, especially TLauncher, here.

Queue

Why is there a queue?

Our system is not infinitely large, and sometimes more users want to play than we have resources for. If our system is full, you have to wait until other players have finished playing and their server got stopped. It's like at the supermarket: If all employees are already serving other customers, you have to wait until it's your turn. That's the only way to keep it fair for everyone.
